## How this works
- Project memory lives in `Projects/{repo-name}/` — see
  `claude/templates/project-skeleton.md`.
- Note types: `research/`, `decisions/`, `bugs/`, `patterns/`, `sessions/`.
- Update the project `_index.md` whenever you add or rename a note.
- Before researching a topic, check `research/` first — never redo captured work.

*Seeded by mac-config (Nix). Safe to edit — Nix won't overwrite existing files.*
